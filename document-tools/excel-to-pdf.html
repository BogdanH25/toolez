⚙️ Server-side processing required

Converting Excel to PDF with correct formatting, charts, and print areas requires a server-side engine. Browsers alone cannot reliably render Excel styles into PDF.

In the meantime, you can convert for free using Microsoft Excel Online → File → Export → PDF or Google Sheets → File → Download → PDF.
